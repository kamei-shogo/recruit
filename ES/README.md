# ES（エントリーシート）管理

このディレクトリにエントリーシートを Markdown ファイルとして保存します。

## ファイルの命名規則

```
<企業名>_<提出日 YYYYMMDD>.md
```

例:
- `Sharp_20260312.md`
- `Toyota_20260401.md`
- `Sony_20260315.md`

## 作成手順

1. `_template.md` をコピーして命名規則に従ってリネームします。
2. テンプレートの各項目を記入します。
3. コミット・プッシュしてリポジトリに保存します。

## GitHub Projects からのリンク方法

GitHub Projects のテーブルで `ESリンク` フィールドに、このディレクトリからの相対パスを入力します。

例: `ES/Sharp_20260312.md`

GitHub 上でリンクをクリックすると Markdown が整形表示されます。
